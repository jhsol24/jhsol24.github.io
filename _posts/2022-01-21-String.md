## String (문자열)
> 변수의 한 종류인 String 변수에 대해 공부한다.

### `length` 문자열의 길이
문자열의 길이를 호출할 때 사용한다.
```
let myString = 'Hello World'
console.log(myString.length)
>> 11
```

### 주의할 점
#### Number + String = String
숫자열과 문자열을 `+` 연산자로 연결하면 문자열로 취급한다.
```
console.log(2 + '2')
>> 22
```
