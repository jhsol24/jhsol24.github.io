## `console.log`

- 디버깅 시 로그 확인용으로 사용

## Variable (변수)
> 데이터를 담을 수 있는 대상
Javascript 에서는 대부분 선언과 할당이 동시에 이루어진다.
> 1. 선언 : 변수를 사용하겠다고 변수 이름을 지정
> 2. 할당 : 변수에 내가 담고자 하는 데이터를 담는 과정

### 변수 타입 종류
1. Primitive type
    - Number, String, Boolean, Undefined, Null
2. Reference type
    - Object, Array, Function
